# Galsenify's docs

Documentation officielle de [Galsenify](https://www.npmjs.com/package/galsenify), une bibliothèque npm complète de données sénégalaises (régions, départements, télécom, recherche, calculs).

## Stack

- [Next.js 15](https://nextjs.org) — Pages Router
- [Tailwind CSS 4](https://tailwindcss.com)
- [Framer Motion](https://motion.dev)
- [Galsenify v1.2.0](https://www.npmjs.com/package/galsenify)

## Fonctionnalités

- Documentation complète des 30 méthodes de galsenify
- Exemples interactifs testables directement dans le navigateur
- Internationalisation FR / EN
- Animations Framer Motion

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000) dans le navigateur.

## Build

```bash
npm run build
npm start
```
