# Galsenify's docs

Official documentation for [Galsenify](https://www.npmjs.com/package/galsenify), a comprehensive npm library for Senegalese data (regions, departments, telecom, search, calculations).

**_Lire en [Français](./public/assets/README.fr.md)_**

## Stack

- [Next.js 15](https://nextjs.org) — Pages Router
- [Tailwind CSS 4](https://tailwindcss.com)
- [Framer Motion](https://motion.dev)
- [Galsenify v1.2.0](https://www.npmjs.com/package/galsenify)

## Features

- Full documentation for all 30 galsenify methods
- Interactive examples testable directly in the browser
- FR / EN internationalization
- Framer Motion animations

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```
